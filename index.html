Comparison of Visualization Tools

In the readme report I summarized my experiences, likes, and dislikes for each visualization tool. However, I wanted to provide some direct comparisons in more direct ways. Therefore, I ranked all of the tools in 5 distinct categories: learnability, file size, customization, hackiness, and visual appeal. Although this is mostly subjective, it's based on my experience making these scatter plots and helps to compare the different tools.

Learnability

This shows the ranking of the tools based on a combination of how easy they were to learn and the quality of their documentation. This is the most important metric in my opinion.

Summary

Based on these results I would say R + ggplot2 is the best visualization tool for creating a bubble chart. With Vega-Lite as a close second place. This suprises me a little because R was the tool I was most afraid to use going in. But considering the graph we are trying to model was also build in R, maybe this makes sense. I definitely enjoyed learning R and found it very easy to use.

There is some bias in this ranking. All of the file sizes are small and that may not be a big factor, although writing all of the R code in a few lines is cool. Additionally, learnability, in my opinion, is the most valuable the metrics I measured. Tableau was incredibly easy to learn, so despite these results, I may advise people inexperienced with coding to use Tableau in the future. It's just very user friendly.
